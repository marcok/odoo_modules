Employee Time Clock

Over- and undertime

Employee Time Clock calculates over- and undertime, this information is displayed and stored in employees timesheets.
The user see the current account balance of the timesheet and the overall account balance from beginning as well on the main timesheet screen. In a new tab "attendance analysis" the totals are breaked down for each day in the timesheet period.
The duty hours are calculated based on the timeschedule assigned to an employee/contract and therefore can be configured on a daily basis.
On days where an approved leave exists, duty hours are set to zero.
After a timesheet is approved, the balance of the timesheet is persisted on timesheet level for performance and to avoid manipulation.
Info: This module does not change the behaviour of attendances nor worked hours on analytic accounts, it adds a new type "duty hours" and makes calculations based on that.

Timesheet generation

For convenience, this screen gives a manger the possibility to generate multiple timesheets for all employees with a certain tag. If you wanna track over- and undertime it is recommended to generate timesheets for the whole year in advanced, usually on a monthly basis.

Import leaves

If you track under- and over-time its important to ignore public holidays in the calculation. This screen allows you to upload public holidays (or any other leave types) with an excel/csv file (1 column with dates), which will be imported as approved leave requests for employees with a certain tag.

Installation

Under Settings -> Human Resources you should activate "Track attendances for all employees".
If users dont have a timezone defined, you get errors on sign in/out.
Security: The module adds a rule, which allows employees to read there own contracts, this shouldn't be a problem, but you need to keep that in mind.
Security: The module adds a rule, which only allows managers to see the timesheets of there subordinates. If you want to have the old behaviour, that a manger can see all timesheets, you may set this rule inactive.

Odoo time clock (iPad & iPhone)

This module is well integrated with our iPhone and iPad apps, certainly it works without the mobile apps as well. It's free up to 3 users and available on iTunes.
Odoo time clock gives the employees the possibility to sign-in/sign-out with a 2D barcode card, e.g. on the entrance of your company where the iPad is mounted, in under 2 seconds.
The app works best with an iPad, but e.g. at an office reception it works also great with an iPhone or even an iPod touch.

Sign in/out with 2d barcode

An employee can sign in/out with holding his 2d code card to the front cam. The check-in process automatically detects if the user is currently signed in or out and executes the following step. The employee gets a quick overview of his over- and undertime balance. Employees don't even need to have an odoo user.

Producing the 2d cards is easy and without additional costs

You can export your employees data from odoo as excel (name [optional] and employee id which is used for the 2d barcode [interger]) and create the stickers by yourself with your printer. Use e.g. the Herma online tool for all herma sticker formats: Herma online tool. The tool allows you to create your desired design inclusive logo and generate a pdf for printing.

Lunch order

If odoo module "Lunch" is installed and items are defined, they are shown after each sign in and the user can choose his favourite lunch. The app can operate in "Lunch order" only mode.
App settings:
- you can define which categories have to be shown.
- you can define a lunch only mode, then no attendance sign in/out will be recorded. This is useful to use the app e.g. for coffee consummation book keeping.
- you can define if the lunch screen only has to be shown for one order per day or of after each sign in.

Company info message

The company can define company wide information messages as a calender entry, tagged with "Station" and the description of the entry is shown on the time clock app for the desired period. The calender entry has to be created with the same user as used for the app.
The calender is checked by the app every 5 minutes, thus new info messages appear in a few minutes.
Your company logo, if available, is displayed too. You can upload your own logo under your company data.

Odoo HR let your employees execute the most important HR tasks mobile, like sign in/out, make leave requests, see a history of attendances and analyse over- and undertime.
The app is available on iTunes.

Attendance analysis

Employees can display an under- and overtime analysis for each timesheet. He see his duty hours based on his contract and his attendances hours and the diff (over- and undertime). This feature is only available if Odoo module "Employee time clock" is installed.
